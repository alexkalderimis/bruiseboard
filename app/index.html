Bruise Board

A configurable dashboard for your code

Repository Statistics Travis Builds Tools

GitHub Repository Statistics

Bruise board can display a number of basic statistics about your code repository.

Milestones

See at a glance how much progress you are making to your open milestones. Configure your bruise board to watch one or more milestones and update progress throughout the day.

Statistics

Get an idea of the overall health of your repo in the form of basic statistics, such as number of open issues, forks, and pull-requests.

Travis Builds

Monitor CI builds on Travis-CI. Watch the status of your builds in real time and get instant feedback on the health of your code.

Bruiseboard tries to show you all the important information to help you see how your builds are proceeding. This includes sub-jobs, so you can tell the eventual state before the entire build is done, as well as getting an idea of how thoroughly your build is broken (or not!)

Tools

Data

This application is built on top of the open GitHub and Travis-CI APIs, which they generously provide to allow you to access data about your code and allow us to build useful secondary tools such as this.

Frameworks

The site is rendered with React.JS from Facebook, with a little help from jQuery, and styled with SemanticUI. This is a mainly client based application, with a small server side component using express and MongoDB.

Services

The application itself is open-source under the MIT license, and available on GitHub. You are most likely accessing this page on Heroku, where a public instance is hosted.
